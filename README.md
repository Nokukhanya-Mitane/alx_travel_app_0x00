# ALX Travel App 0x00 — Database Modeling & Seeding

This project implements database models, serializers, and a data seeder for a travel booking application.

## ✔ Features

### **1. Database Models**
- Listing
- Booking
- Review  
Located in: `listings/models.py`

### **2. Serializers**
- ListingSerializer (with nested Bookings)
- BookingSerializer  
Located in: `listings/serializers.py`

### **3. Database Seeder**
Run:
```bash
python manage.py seed
